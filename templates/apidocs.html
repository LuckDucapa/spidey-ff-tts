API Documentation

Integrate SonicTTS into your apps via HTTP Requests.

1. Specific Voice Match

Use a character name like 'Adri', 'Guy', 'Aria'.

GET /tts?text=Hello World&voice=Adri

2. Smart Filtering

Filter by Language, Gender, and optional Country.

GET /tts?text=Namaste&lang=hi&gender=Female

3. Speed/Rate Control

Control speaking speed (e.g., +50%, -20%). Default is +0%.

GET /tts?text=Too Fast!&voice=Guy&rate=-30%

4. Country/Accent Specific

Target a specific country code (e.g., GB, IN, US).

GET /tts?text=Cheers mate&lang=en&country=GB
